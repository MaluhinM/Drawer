В папке <hotkeys> хранятся горячие клавиши для работы с программой.
В случае надобности, значения параметров можно изменить.
Для изменения параметров нужно вписать в определенный текстовый файл нужное Вам сочетание клавиш.
В зависимости от выбранного файла изменится определенный параметр.
Список параметров по умолчанию указан снизу:

В папке <base> находятся файлы, отвечающие за обычное движение исполнителя Черепаха в программе:
forward - движение вперед (w)
backward - движение назад (s)
left - поворот влево на 90° (a)
right - поворот вправо на 90° (d)

В папке <shift> находятся файлы, отвечающие за движение с использованием клавиши Shift:
shift_forward - усиленное движение вперед (Shift + w)
shift_backward - усиленное движение назад (Shift + s)
shift_left - поворот влево на 45° (Shift + a)
shift_right - поворот вправо на 45° (Shift + d)

В папке <ctrl> находятся файлы, отвечающие за движение с использованием клавиши Ctrl:
ctrl_forward - слабое движение вперед (Ctrl + w)
ctrl_backward - слабое движение назад (Ctrl + s)
ctrl_left - поворот влево на 5° (Ctrl + a)
ctrl_right - поворот вправо на 5° (Ctrl + d)

В папке <figures> находятся файлы, отвечающие за рисование определенных фигур исполнителем:
figure_circle - круг (1)
figure_snowflake - снежинка (2)

В папке <other> находятся файлы, отвечающие за прочие возможности программы:
exit - завершение работы программы <Esc>
reset_heading - сброс поворота исполнителя <Del>
reset_position - сброс позиции исполнителя <Shift + Del>
clear - сброс холста <Ctrl + Del>
reset - полный сброс <Shift + Ctrl + Del>
reset_pencolor - сброс цвета пера <F7>
reset_bgcolor - сброс цвета заднего фона <F8>
fill - запуск/окончание заливки <F4>
prtsc - снимок экрана <Ctrl + F2>
prtsc_dir - перейти в папку со снимками экрана <Shift + Ctrl + F2>
pencolor - изменить цвет пера <F5>
bgcolor - изменить цвет заднего фона <F6>
size- - уменьшить размер пера <->
size+ - увеличить размер пера <+>
undo - отмена действия <Ctrl + z>
pen - поднятие/опускание 'ручки' исполнителя <Tab>
hide - скрыть/показать исполнителя <F1>

Учтите, что в случае указания недопустимого значения клавиш,
программа откажется работать и уведомит об этом окном с указанием на ошибку.
Для устранения проблемы следует откатить изменения к первоначальному виду.
Универсальный метод решения проблем - переустановка программы.
