Ниже представлено основное управление по умолчанию, без каких либо изменений в настройках программы.
Клавиши или сочетание клавиш записано в угловых скобках ('<' и '>').
Напротив указано действие, которое должна совершить программа при использовании той или иной клавиши или сочетании клавиш.
В мини-игре присутствуют следующие функции:

- Основное передвижение:
-> Осуществляется нажатием кнопок: <w>, <a>, <s>, <d>. Где...
--> <w> - Движение вперед
--> <a> - Движение влево
--> <s> - Движение назад
--> <d> - Движение вправо

- Передвижение с нажатой клавишой Shift:
-> Используются те же клавиши, что и для основного передвижения, но с дополнительно нажатой клавишей Shift

- Передвижение с нажатой клавишой Ctrl:
-> Используются те же клавиши, что и для основного передвижения, но с дополнительно нажатой клавишей Ctrl

- Прочие функции:
-> <Esc> - Завершение работы программы
-> <Del> (<Delete>) - Сброс поворота
-> <Shift + Del> - Сброс позиции исполнителя
-> <Ctrl + Del> - Очистка холста
-> <Shift + Ctrl + Del> - Полный сброс
-> <F7> - Сброс цвета пера
-> <F8> - Сброс цвета заднего фона
-> <F4> - Запуск/окончание заливки
-> <Ctrl + F2> - Снимок экрана
-> <Shift + Ctrl + F2> - Перейти в папку со снимками экрана
-> <F5> - Изменить цвет пера
-> <F6> - Изменить цвет заднего фона
-> <-> - Уменьшить размер пера
-> <+> - Увеличить размер пера
-> <Tab> - Поднятие/опускание 'ручки'
-> <F1> - Скрыть/показать исполнителя
-> <Ctrl + z> - Отмена действия

- В программе также используются 'фигуры', которые являются внутренними изменяемыми алгоритмами программы.
Они выполняют определенный порядок действий, который обычно направлен на рисование определенных фигур.
Фигуры можно создавать самому, а в программе по умолчанию уже есть 4 в виде пакетов данных, которые нужно установить перед использованием:
-> <1> - Круг
-> <2> - Снежинка
-> <3> - Квадрат
-> <Shift+3> - Квадрат со смещением
